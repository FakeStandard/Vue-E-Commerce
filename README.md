# Vue-E-Commerce
使用前後端分離開發模式來 Vue 出一個電商

:rocket: Technology <br>
Front end：Vue、Vue-router、Element-UI、Axios、Echarts <br>
Back end：Node.js、Express、Jwt、MySql、Sequelize <br>
